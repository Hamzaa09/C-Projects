# C-Projects

A collection of C programs built while learning core programming concepts.

## Featured Project: Tic Tac Toe

A console-based Tic Tac Toe game supporting two-player and single-player (vs Computer) modes.

![Main Menu](./Project%203%20(TicTacToe)/screenshots/1.png)
![Player vs Player](./Project%203%20(TicTacToe)/screenshots/2.png)


**Features**
- Two game modes: Player vs Player, Player vs Computer
- Randomized computer moves
- Input validation for out-of-range or already-taken cells
- Win and draw detection across all rows, columns, and diagonals

**Build & Run**
```bash
gcc TicTacToe.c -o TicTacToe
./TicTacToe
```
Built and tested on Windows (MinGW/GCC), since it uses `conio.h` and `system("cls")`.
